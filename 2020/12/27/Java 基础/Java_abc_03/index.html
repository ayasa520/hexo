Java 基础(三) 对象数组

发表于2020-12-27|更新于2022-05-01|教程
|阅读量:

对象数组与基础类型的数组如int []是不同的。对象数组的每个元素都是对象的管理者，而非对象的本身。

当通过创建了一个String类型的数组时，这个数组的每一个“格子”里都是String类型的管理者。此时该数组每一个元素管理的都是空的，即null。
